with accessor keyword

The following tasteless rust code

ctx.line_to(offset_x, offset_y);
            ctx.stroke();
            ctx.begin_path();
            ctx.move_to(offset_x, offset_y);

can be beautified in angle as

with context do 
  stroke
  begin path
  move to offset.x offset.y
end
